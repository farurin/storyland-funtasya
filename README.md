# React + Vite

How to install book funtasya project:

1. Git clone

git clone https://github.com/Okaarunn/book-funtasya.git

2. Enter directory

cd book-funtasya

3. Install depedency

npm install

4. Run project

npm run dev

2. UI: https://flowbite-react.com/docs/getting-started/cli
